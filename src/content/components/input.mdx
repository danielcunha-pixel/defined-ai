---
title: Input
description: Text input field for capturing user data in forms. Supports various states, labels, and helper text.
---

The Input component is a styled text field for capturing user data. It supports various states including default, disabled, and invalid.

## When to use

- Collecting text-based user input (names, emails, searches)
- Form fields that require single-line text entry
- Search interfaces

Combine with labels and helper text for accessible, complete form fields. Icons can be added using relative positioning.
